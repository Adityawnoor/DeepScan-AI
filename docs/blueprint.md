# **App Name**: DeepScan AI

## Core Features:

- Deepfake Detection Service: Utilize a Claude AI-powered tool to analyze uploaded images, videos, or audio (initially focusing on images in MVP) for deepfake authenticity, returning a confidence score and verdict (real/fake).
- Intuitive Media Upload: Provide a user-friendly interface with drag-and-drop functionality and visual previews for various media file types, along with direct webcam capture capability.
- Dynamic Analysis Result Display: Visually present the AI's detection confidence, the 'real' or 'fake' verdict, and a simulated area of interest overlay to highlight potentially manipulated regions.
- Recent Detection History: Display a lightweight, paginated log of recently analyzed files, including their status, and detection outcomes, stored in the specified database.
- Theme Customization Toggle: Allow users to switch between dark and light modes for an enhanced and personalized user interface experience.

## Style Guidelines:

- Predominantly light for a clean, professional, and accessible aesthetic. The background is a crisp off-white (#F8F9FA).
- Primary accent color: A calming, medium blue (#007BFF) for main interactive elements and brand emphasis, conveying reliability and professionalism.
- Secondary accent color: A subtle, lighter blue (#6C757D) for secondary actions, text, and less prominent highlights, ensuring visual harmony.
- Headline and body text font: 'Inter' (sans-serif) for its modern, clear, and objective appearance, ensuring high legibility for data presentation.
- Icons: Employ clean, minimalistic vector icons, as suggested by 'lucide-react' in the project requirements, maintaining consistency and professionalism across the application.
- Layout: A spacious, organized, and data-centric layout, reminiscent of professional dashboards, with a focus on clear information hierarchy and user workflow.
- Animation: Subtle and functional animations, such as smooth transitions for file uploads, result loading indicators, and page navigation, to provide seamless user feedback without distraction.
